NFL Game Prediction in Excel

By: Brindyn Schultz & David Abrutis


Problem & Solution

With how uncertain and risky sports betting can end up being, this application will give predictions based on prior calculations that can help the end user make better decisions based on the predictions given. This is important because with how popular sports betting is, and how important losing or gaining money is, being able to reference a calculated prediction with no biases and only numbers, it can allow for safer bets, and consequently, better profits. Predictions will be weighted and will take into account recent games that will allow the predictions to stay up to date, game to game.

Application Overview

The application will contain a user-friendly interface on a single spreadsheet that calculates and updates game-predicting probabilities for the NFL season as it occurs.

Technology Overview

  • Microsoft VBA
  • Microsoft Excel (Google Sheets)
  • Massey Method Predictive Model
  • Colley Method Predictive Model
